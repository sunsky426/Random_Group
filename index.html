True Random Group Generator

Have you considered how predictable machines like computers generate randomness? The simple answer is they don't! Most random generator programs are pseudo-random, which means they generate numbers in a predictable fashion using a mathematical formula. These formula generator numbers are very close to random, but not truly random and can be predicted through various methods. True randomness, on the other hand, refers to randomness generated not by a predictable computer algorithm, but by nature, usually through the monitoring of acoustic and thermal noises. This website is an a true random group generator that anyone can use to generate groups for schools or clubs in a truly random fashion.

This website does not track atmospheric noises ourselves, but thankfully, the much more famous and bigger true random website Random.org does. This website operates by requesting random numbers from Random.org, which we then use to generate random groups.


Instructions:

  1. Input the names you're going to group under the names field. Each name needs to be separated by a new line.
  2. Select the number of groups you need
  3. Click GO
  4. Wait for the results to appear under output
